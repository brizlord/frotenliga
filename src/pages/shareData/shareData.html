Compartir Datos


Esta opción es para compartir los datos que se van guardando en la aplicación y así poderlo distribuir entre todos. Cuando se exporta la información, se hace a una carpeta en el almacenamiento del dispositivo llamada Frotenliga y dentro se encuentra el fichero que se pasa de dispositivo a dispositivo (Zapya, Bluetooth, NFC, etc.).
Desde el botón importar se llega hasta la ruta donde se encuentra el fichero y se selecciona.
